漏洞利用

使用 irc 客户端(hexchat和irssi都可以)访问6697 端口，可以看到连接信息中返回的“Unreal3.2.8.1”

通过检索，找到exp，修改脚本并执行

$ python3 exploit.py 10.10.10.117 6697  -payload python

成功接收到shell
